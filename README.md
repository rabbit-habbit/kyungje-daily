# sonkyungje-daily

손경제 Daily Brief 자동화 보고서

## 소개

본 저장소는 손경제(손에 잡히는 경제) 콘텐츠를 일 단위로 요약·정리한 Daily Brief 자동화 보고서를 보관합니다.

## 구조

- `docs/` — GitHub Pages 배포용 정적 사이트 디렉터리
  - `index.html` — Daily Brief 메인 페이지

## GitHub Pages

GitHub Pages는 `main` 브랜치의 `/docs` 폴더에서 서빙됩니다.
저장소 **Settings → Pages**에서 `Branch: main / Folder: /docs`로 설정해 활성화하세요.
